Biblioteka Matplotlib¶

Biblioteka matplotlib jest inspirowaną matlabem biblioteką do tworzenia wykresów

Instalacja¶

W najprostszej wersji (takiej w której matplotlib współpracuje z ipython notebookiem i pozwala zapisywać wykresy na dysk) starczy:

pip install numpy

pip install matplotlib

Bardziej zaawansowanej wersji nie chcę omawiać :)

Uruchomienie matplotliba¶

By połączyć matplotliba z ipython notebookiem starczy:

In [9]: 
%matplotlib inline
from matplotlib import pylab

Od teraz działają wbudowane polecenia do wyświetlania wykresów:

In [10]: 
pylab.plot(range(10))
Out[10]:
[<matplotlib.lines.Line2D at 0x7f793935a668>]

Filozofia matplotliba¶

W matplotlibie istnieje pojęcie "aktualnie" tworzonego wykresu. Można na niego:

  • Dodawać linie
  • Dodawać osie
  • Dodawać legendę
  • ....

Przy połączeniu z ipython notebookiem wszystkie własności aktualnego wykresu są czyszczone po wykonaniu komórki

In [17]: 
lista = list(range(10))
pylab.plot(lista, label="Seria A")
pylab.plot(lista[::-1], label="Seria B")
pylab.legend()
pylab.xlabel("Oś X")
pylab.ylabel("Oś Y")
pylab.title("TYTUŁ")
Out[17]:
<matplotlib.text.Text at 0x7f7939046cc0>

Polecenie clf kasuje aktualnie tworzony wykres:

In [20]: 
lista = list(range(10))
pylab.plot(lista, label="Seria A")
pylab.savefig("fig-a.png")
pylab.clf()

pylab.savefig("fig-b.png")

Wykresy z niepewnościami:¶

In [34]: 
import numpy as np
x = np.arange(0, 100)
data = np.random.random_integers(10, 1000, (100))
data.sort()
stddev = np.sqrt(data)
xerr = np.ones_like(x)
pylab.errorbar(x, data, xerr=xerr, yerr=stddev)
pylab.xlim(0, 100)
Out[34]:
(0, 100)

2. Brak polskich znaków:¶
In [39]: 
pylab.plot(range(10))
pylab.title("Zażółć gęślą jaźń")

<matplotlib.text.Text at 0x7f7938facb00>

Rozwiązanie: Zmienić czcionkę (domyślna nie musi mieć polskich znaków)

In [40]: 
import matplotlib
matplotlib.rc('font', family='Arial')
In [41]: 
pylab.plot(range(10))
pylab.title("Zażółć gęślą jaźń")
