Biblioteka Matplotlib¶

Biblioteka matplotlib jest inspirowan matlabem bibliotek do tworzenia wykresw

Instalacja¶

W najprostszej wersji (takiej w ktrej matplotlib wsppracuje z ipython notebookiem i pozwala zapisywa wykresy na dysk) starczy:

pip install numpy

pip install matplotlib

Bardziej zaawansowanej wersji nie chc omawia :)

Uruchomienie matplotliba¶

By poczy matplotliba z ipython notebookiem starczy:

In [9]: 
%matplotlib inline
from matplotlib import pylab

Od teraz dziaaj wbudowane polecenia do wywietlania wykresw:

In [10]: 
pylab.plot(range(10))
Out[10]:
[<matplotlib.lines.Line2D at 0x7f793935a668>]

Filozofia matplotliba¶

W matplotlibie istnieje pojcie "aktualnie" tworzonego wykresu. Mona na niego:

  • Dodawa linie
  • Dodawa osie
  • Dodawa legend
  • ....

Przy poczeniu z ipython notebookiem wszystkie wasnoci aktualnego wykresu s czyszczone po wykonaniu komrki

In [17]: 
lista = list(range(10))
pylab.plot(lista, label="Seria A")
pylab.plot(lista[::-1], label="Seria B")
pylab.legend()
pylab.xlabel("OX")
pylab.ylabel("O Y")
pylab.title("TYTU")
Out[17]:
<matplotlib.text.Text at 0x7f7939046cc0>

Polecenie clf kasuje aktualnie tworzony wykres:

In [20]: 
lista = list(range(10))
pylab.plot(lista, label="Seria A")

Wykresy z niepewnościami:¶

In [34]: 
import numpy as np
x = np.arange(0, 100)
data = np.random.random_integers(10, 1000, (100))
data.sort()
stddev = np.sqrt(data)
xerr = np.ones_like(x)
pylab.errorbar(x, data, xerr=xerr, yerr=stddev)
pylab.xlim(0, 100)
Out[34]:
(0, 100)
